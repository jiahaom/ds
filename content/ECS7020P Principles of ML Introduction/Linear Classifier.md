# Boundaries
- linear equation: $w^Tx=0$
	- extended vector / predictors: $x=[1, x_1,x_2,...]^T$
	- coefficients vector: $w=[w_0, w_1,w_2...]$
	- e.g. 2D
		- $x_2 = -\frac{w_1}{w_2}x_1-\frac{w_0}{w_2}$


# Definition
- $w^Tx>0$: one side of the boundary
- $w^Tx<0$: other side of the boundary
